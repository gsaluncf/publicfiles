Seminar Agenda

1.5-day schedule with session breakdown

What You Will Learn

By the end of this seminar, you will be able to:

  • Design a database schema for financial data with temporal attributes
  • Implement Geometric Brownian Motion to simulate stock prices
  • Sample from actuarial life tables to model uncertain lifespans
  • Run 10,000+ Monte Carlo simulations and store results in DuckDB
  • Write SQL queries to compute expected value, VaR, and win rates
  • Find decision boundaries where the optimal strategy changes

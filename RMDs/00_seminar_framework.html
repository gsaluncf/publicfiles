Seminar Framework

Stochastic Tax-Optimization & Portfolio Simulation

1.5-Day Intensive for Data Science Graduate Students

Seminar Thesis

Retirement wealth management is a stochastic optimization problem with no closed-form solution. Students will build a Monte Carlo simulation engine to quantify "Tax Alpha" - the value generated by optimizing financial decisions around Required Minimum Distributions and the Step-up in Basis tax rule.

Learning Objectives

Domain Objective Assessment
Probabilistic Modeling Implement GBM for market simulation; apply survival analysis for mortality modeling Working simulation code
Database Architecture Design temporal schemas for tax lot tracking; understand lookup vs. transactional tables Schema design exercise
Monte Carlo Methods Execute 10K+ simulations; aggregate results using SQL Query results
Decision Science Quantify Tax Alpha; find decision boundaries between strategies Final visualization

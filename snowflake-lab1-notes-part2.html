4. Exploring Existing Data

4.1 Show Available Tables

First, let's see what tables are available in your current context (lab1.sql line 207):

SHOW TABLES;

üí° Important Note

SHOW commands are metadata operations - they don't use your virtual warehouse. Notice the gray dot next to your warehouse name indicates it's suspended.

4.2 Query Snowbear Air Data

Let's explore the airline data with these queries (lab1.sql lines 218-241):

Simple Member Data Query:

SELECT *
FROM members
LIMIT 10;

Top Customers by Account Balance:

SELECT TOP 5
    c.c_lastname,
    c.c_firstname,
    c.c_acctbal
FROM snowbearair_db.promo_catalog_sales.customer c
ORDER BY c.c_acctbal DESC;

Customer Orders with JOIN:

SELECT
  c_firstname,
  c_lastname,
  o_orderkey,
  o_totalprice
FROM
  snowbearair_db.promo_catalog_sales.orders
JOIN
  snowbearair_db.promo_catalog_sales.customer
ON
  o_custkey = c_custkey
ORDER BY o_totalprice DESC
LIMIT 10;

Check your understanding

  • Why does the last query use full table paths (database.schema.table)?
  • What happens to your warehouse when you run these queries?
  • What's the difference between LIMIT and TOP?

5. Creating Your Own Objects

5.1 Create Your Database

Create and use your personal database (lab1.sql lines 258-260):

CREATE DATABASE WOODCHUCK_db;
USE DATABASE WOODCHUCK_db;

5.2 Create Your Schema

Create and use your personal schema (lab1.sql lines 265-267):

CREATE SCHEMA my_schema;
USE SCHEMA my_schema;

5.3 Set Default Context

Configure your user defaults for future sessions (lab1.sql lines 275-279):

ALTER USER WOODCHUCK
SET default_warehouse=WOODCHUCK_wh
    default_namespace=WOODCHUCK_db.public
    default_role=training_role;

7. Key Takeaways

What You've Learned

  • Object Creation: You can create database objects via Snowsight UI or SQL commands
  • Object Browser: Use the worksheet object browser to view and navigate database objects
  • Context Components: Role, schema, database, and virtual warehouse define your session context
  • Context Setting: Set context via Snowsight UI or SQL statements
  • Organization: Create folders to save and organize worksheets

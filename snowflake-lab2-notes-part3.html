4. Add Tiles to Dashboard

üí° Dashboard Tiles

There are several ways to add tiles to a dashboard:

  • Create new: Build a tile from scratch
  • Duplicate: Copy an existing tile and modify its query
  • Add chart: Convert numeric data to a graph visualization

We'll use the third method to create a chart from our existing data.

4.1 Create Chart Tile

  1. Edit existing tile: Click the ellipsis (‚ãØ) in the upper right corner of your data tile
  2. Select "Edit Query"
  3. You'll return to the original query with results displayed
  4. Create chart: Click the "Chart" button below the SQL and above the query results
  5. A line graph will be displayed by default
  6. Return to dashboard: Click "Return to WOODCHUCK Gross Sales" in the upper left

4.2 Arrange Dashboard Layout

  1. Reposition tiles: Click, hold, and drag the graph tile
  2. Position above data: When you see a blue cursor above the data tile, release the mouse
  3. Your dashboard should now show the graph above the data table

5. Share Your Dashboard

5.1 Configure Sharing

  1. Open sharing dialog: Click the "Share" button in the upper-right corner
  2. The dialog allows you to search for users and invite them to view/use the dashboard
  3. Add user: Enter "Instructor1" as the user to share with
  4. Set permissions: Click the down arrow next to the username
  5. Select "View + run" permissions
  6. Complete sharing: Click "Done"

üîê Permission Levels

  • View: Can see the dashboard but not interact with it
  • View + run: Can view and execute queries in the dashboard
  • Edit: Can modify the dashboard structure and queries

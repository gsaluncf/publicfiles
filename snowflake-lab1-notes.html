2.2 Understanding Worksheet Components

Each worksheet is a session connected to Snowflake with its own context:

  • Role: Determines permissions and object visibility
  • Virtual Warehouse: Compute resource for running commands
  • Database & Schema: Default location for object operations

üí° Context Example

If your context is set to WOODCHUCK_DB.MY_SCHEMA, then:

  • SELECT * FROM members looks in your current schema
  • SELECT * FROM other_db.other_schema.table specifies full path

  WAREHOUSE_SIZE = XSmall

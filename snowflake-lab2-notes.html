Lab 2: Visualizations in Snowsight

Objective: Learn to use Snowsight's visualization features, contextual statistics, and dashboard creation for Snowbear Air's sales analysis.

Scenario: Snowbear Air wants a year-by-year summary of gross sales with graphs and dashboards using the PROMO_CATALOG_SALES schema.

1. Lab Setup

1.1 Create Workspace

  1. Create a new folder called "Visualizations"
  2. Create a worksheet inside the Visualizations folder
  3. Load the SQL file: 02-visualizations-in-snowsight.sql
  4. Rename the worksheet to "Dashboard Data"

1.2 Set Context

Execute these statements to set up your environment:

USE ROLE TRAINING_ROLE;
USE SCHEMA SNOWBEARAIR_DB.PROMO_CATALOG_SALES;
CREATE WAREHOUSE IF NOT EXISTS WOODCHUCK_wh;
USE WAREHOUSE WOODCHUCK_WH;

1.3 Initial Query

Run this query to get year-by-year gross sales data:

SELECT 
  YEAR(o.o_orderdate) AS year,
  SUM(l.l_extendedprice) AS sum_gross_revenue
        
FROM
        CUSTOMER C
        INNER JOIN ORDERS O ON C.C_CUSTKEY = O.O_CUSTKEY
        INNER JOIN LINEITEM L ON O.O_ORDERKEY = L.L_ORDERKEY

GROUP BY year
ORDER BY year;

  4. Ensure no decimal values appear in the YEAR column
